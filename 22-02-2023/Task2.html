HTML Formatting

HTML Formatting is a process of formatting text for better look and feel.
HTML provides us ability to format text without using CSS.

There are many formatting tags in HTML.
These tags are used to make text bold, italicized, or underlined.

In HTML the formatting tags are divided into two categories:

  • Physical Tag: These tags are used to provide the visual appearance to the text.
  • Logical Tag: These Tags are used to add some logical swmentic value to the text

Following are the formatting elements in HTML:
  • <b> : This is a physical tag, which is used to bold the text written between it

    • Syntax

                        <b>This will be bold</b>

    Example : This text is Bold

  • <i> : This is a physical tag which is used to make text italic.

    • Syntax

                        <i>This will be italic</i>

    Example : This text is italic

  • <strong> : This is a logical tag, which tells the browser that the text is important.

    • Syntax

                        <strong>This will be strong </strong>

    Example : This text is Strong

  • <em> : This is a logical tag which is used to display content in italic.

    • Syntax

                        <em>This will be strong </em>

    Example : This text is emphesised

  • <mark> : This tag is used to highlight text.

    • Syntax

                        <mark>This will be highlighted </mark>

    Example : This text is highlighted

  • <u> : This tag is used to underline the text written between it.

    • Syntax

                        <u>This will be underlined </u>

    Example : This text is underlined

  • <sup> :It displays the content slightly above the normal line.

    • Syntax

                        3<sup>2<sup> = 9

    Example : 32 = 9

  • <sub> : It displays the content slightly below the normal line.

    • Syntax

                        H<sub>2<sub>O

    Example : H2O

  • <del> : This tag is used to display the deleted content.

    • Syntax

                        Hello world <del>This text is deleted </del>

    Example : Hello world This text is deleted

  • <ins> : This tag displays the content which is added.

    • Syntax

                        Hello world <ins>This text is added </ins>

    Example : Hello world This text is added

  • <big> : This tag is used to increase the font size by one conventional unit.

    • Syntax

                        Hello world <big>This text is big</big>

    Example : Hello world This text is big

  • <small> : This tag is used to decrease the font size by one unit from base font size.

    • Syntax

                        Hello world <small>This text is big</small>

    Example : Hello world This text is small
