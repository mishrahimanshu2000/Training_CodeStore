Selectors in CSS

The style rule of CSS is made of 3 parts

Selector

A selector is HTML tag at which the style is applied. This could be any tag like : <h2>, <table> etc.

Property

A Property is a type of attribute of HTML tag at which a style will be applied. HTML attribute is converted into CSS Property. This could be color, border, etc.

Value

Value are assigned to Property. For example color property have the Value either red or #f1f1f1 etc.

You can put CSS style rule syntax as follows : 
   selector{ property : value }
Example: 
   table{border: 1px solid black}

We can define selectors in various different ways based on our comfort. Few of them are:

1. Element/Type Selectors:

This is the selector in which the Element or tag is selected and styled.
Example : To give a color to all level 1 heading: 
                h1{
                    color : red
                }

2. Id Selector:

we can define the style rules based on the id attribute of the Element. All the Elements having that id will be formatted according to defined rules. 

                    #black{
                        color : #000000
                    }

3. Class Selector:

we can define style rule based on class attribute of the Element. All element having that class will be formatted according to defined rule. 

                    .black{
                        color : black
                    }
the above rule will render the content in black for every element with class attribute set to black in our HTML document.

To make it bit more particular 
                    h1.black{
                        color : #000000
                    }
this rule will render the content in black for only <h1> Elements with class attribute set to black

4. Universal Selector

Rather then selecting elements of specific type, the universal selector quite simply matches the name of any element type: 

                    *{
                        color : black
                    }
