Selector Grouping

We Can apply a style to many selectors of we like. Just spprate the selectors with a comma(,).
Example 

                h1, h2, h3 {
                    color: red;
                    font-weight: normal;
                    letter-spacing: .4em;
                    margin-bottom: 1em;
                }
This define style rule will be applicable to h1, h2 and h3 as well, the order does not matter for grouping the selectors.
