Use of <a> tag

The <a> tag defines a hyperlink, which is used to link from one page to another.

The most important attribute of the <a> element is the href attribute,
which indicates the link's destination.

By default, links will appear as follows in all browsers:

  • An unvisited link is underlined and blue
  • A visited link is underlined and purple
  • An active link is underlined and red

Few of the important attributes are

  • href
    The href attribute specifies the URL of the page the link goes to.
    If the href attribute is not present, the <a> tag will not be a hyperlink.
    eg: <a href="https://www.w3schools.com/tags/att_a_href.asp">click here </a>
    click here to know more about href

  • target
    The target attribute specifies where to open the linked document
    Syntax : <a target="_blank|_self|_parent|_top|framename"> </a>
    click here to know more about target
    The above link will be open in new tab of the browsers

  • download
    Download file when clicking on the link (instead of navigating to the file).
    Syntax : <a href="/images/myw3schoolsimage.jpg" download="w3logo"> </a>

    click here to know more about download

Some other attributes of a tag are :

  • hreflang
    The hreflang attribute specifies the language of the document in the link.
    Syntax : <a href="https://www.w3schools.com" hreflang="en">W3Schools</a>
    click here to know more

  • media
    The media attribute specifies what media or device the linked document is optimized for.
    This attribute is used to specify that the target URL is designed for special devices (like iPhone), speech or print media.
    Syntax : <a href="att_a_media.asp?output=print"media="print and (resolution:300dpi)"> Open media attribute page for print.</a>
    click here to know more

  • ping
    The ping attribute specifies a list of URLs to be notified if the user follows the hyperlink.
    This attribute is useful for monitoring/tracking.
    Syntax : <a href="https://www.w3schools.com/html" ping="https://www.w3schools.com/trackpings"> </a>
    click here to know more

  • rel
    The rel attribute specifies the relationship between the current document and the linked document.
    Syntax : <a rel="nofollow" href="http://www.functravel.com/">Cheap Flights </a>
